鼠标悬停添加类

Bootstrap下拉菜单默认需要通过点击，才可以打开，但是实际使用悬停代替点击能减少一步操作，更人性化。

所以对bootstrap下拉菜单进行改写-。-

经过分析出，其实就是在.dropdown 后面加多一个open类

用JQ写应该很简单

var dropdownList = $(.dropdown);
dropdownList.mouseover(function(){
    $(this).addClass('open');
}).mouseout(function(){
    $(this).removeClass('open');
});

我又想试着用原生去实现一下，增加类和删除类

一个蠢萌的方法

var dropdownLi = document.getElementsByClassName("dropdown");
for(i=0; i<dropdownLi.length; i++){
  var aLi = dropdownLi[i];
  aLi.onmouseover = function(){ 
  this.setAttribute("class","dropdown open");
  }
  aLi.onmouseout = function(){ 
  this.setAttribute("class","dropdown");
  }
 }

其实应该通过动态添加类删除类实现的
